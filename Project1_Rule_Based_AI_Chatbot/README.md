# DecodeLabs AI Project 1 – Rule-Based AI Chatbot

## Project Overview

This project was developed as Project 1 of the DecodeLabs Artificial Intelligence Internship.

It is a Rule-Based AI Chatbot built using Python. The chatbot responds to predefined user inputs using a Python dictionary and demonstrates fundamental programming concepts such as control flow, decision-making, user input handling, and rule-based logic.

## Features

- Rule-Based AI Chatbot
- Dictionary-Based Response Lookup
- Continuous Conversation Loop
- Input Sanitization using `strip()` and `lower()`
- Multiple User Intents
- Default Fallback Response
- Exit Command

## Technologies Used

- Python 3
- Visual Studio Code

## How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Open the terminal.
4. Run the following command:

```bash
python chatbot.py
```

5. Start chatting with the chatbot.
6. Type `exit` to close the application.
