# DecodeLabs AI Internship - Project 3

## Project Title

**AI Recommendation Logic**

## Project Overview

This project was developed as part of the DecodeLabs Artificial Intelligence Internship.

The project implements a simple AI Recommendation System using rule-based logic. It takes the user's interests as input, matches them with predefined categories, and displays personalized recommendations.

## Features

- Displays available interest categories.
- Takes user interest as input.
- Matches user preferences using rule-based recommendation logic.
- Displays personalized recommendations.
- Handles invalid user input.
- Simple and interactive command-line interface.
